# Быстрая установка системы тегов

## Шаги установки

### 1. Выполните SQL скрипты в Supabase
В SQL Editor выполните файлы в порядке:

1. `add_tag_system.sql` - добавляет поле тега и функции
2. `cleanup_functions.sql` - очищает старые функции (если есть ошибки)
3. `update_functions_with_tags.sql` - создает новые функции с поддержкой тегов

### 2. Проверьте установку
Выполните `test_tag_system.sql` для проверки.

### 3. Перезапустите бота
Код уже обновлен, просто перезапустите бота.

## Что получите
- ✅ Автоматические теги #irl_1, #irl_2, ...
- ✅ Анонимность в топах и профитах
- ✅ Команда `/changetag` для смены тега
- ✅ Обновленный профиль с тегом

## Проверка работы
```
/me - покажет ваш тег
/top - покажет топ с тегами
/changetag #my_tag - сменит тег
```

## Если что-то не работает
1. Проверьте что все SQL скрипты выполнились без ошибок
2. Убедитесь что поле `user_tag` добавлено в таблицу `users`
3. Перезапустите бота
